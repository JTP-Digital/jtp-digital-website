# JTP Digital — Website

Marketing landing page for JTP Digital, a digital marketing agency for local
service businesses (trades, cleaners, pest control, landscapers and more).

## Stack

- Single `index.html` — static HTML with inline CSS and vanilla JS, no build step.
- **Netlify Forms** for the quote / visibility-check enquiry form.
- Hosted on **Netlify**, custom domain `jtpdigital.com.au`.

## Develop

Just open `index.html` in a browser. No dependencies, no build.

> Note: the enquiry form only captures submissions on the deployed Netlify site,
> not when opening the file locally.

## Deploy

Connected to Netlify — every push to `main` triggers an automatic deploy.
